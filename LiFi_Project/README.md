# 🌟 LiFi Communication System

A LiFi Communication System using Arduino for wireless data transmission through visible light.

## 📡 Features
- Wireless data transfer using light
- Arduino-based transmitter and receiver
- Fast and secure communication

## ⚙️ Components
- Arduino boards
- LED (Transmitter)
- Light sensor (Receiver)

## 🚀 Project Type
Academic / Embedded Systems Project
